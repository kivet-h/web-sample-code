# umi project

## 快速开始

安装依赖

```bash
yarn
```

运行项目

```bash
# 本地环境运行
yarn start

# 以测试环境运行
yarn start:dev

# 以生产环境运行
yarn start:prod
```

项目代码打包

```bash
# 打包测试环境代码
yarn build

# 打包正式环境代码
yarn build:prod
```

本地校验

```bash
yarn serve
```

代码提交

```bash
yarn commit

git push
```

项目代码提交要求需遵循代码提交规范，另为了代码提交便捷，项目配置了相关的提交命令。
